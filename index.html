BASIC CLASS

정규수업의 첫 과정으로 톱질연습을 시작으로 공구함 제작, 오일 마감 방법, 자유작품 2가지를 통해 못을 사용하여 가구를 제작하는 방법을 배웁니다.

수업기간 : 2개월(주 1회 3시간 / 8주 과정)

EXPERT CLASS

기초반 수료 후 전문적인 목공 기술 및 각종 짜맞춤 기술을 배우는 과정입니다. 짜맞춤을 활용한 사개함, 스툴 2개를 제작하면서 목공기계와 수공구 사용법을 익힙니다.

수업기간 : 4개월(주 1회 3시간 / 16주 과정)

HOBBY CLASS

기초반, 전문반을 수료 후 취미활동으로 가구 제작을 배우는 수업입니다. 처음부터 끝까지 직접 디자인하고 나무를 골라 가구 또는 소품을 제작합니다.
